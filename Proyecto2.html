Proyecto Independiente

Servidor Web y SFTP Personal

El objetivo de este proyecto fue crear un servidor doméstico de bajo consumo capaz de funcionar 24/7. Utilicé una Raspberry Pi 3 como hardware base, instalando sobre ella el sistema operativo Debian (Raspberry Pi OS) sin entorno gráfico para optimizar el rendimiento y gestionarlo todo mediante comandos de terminal.

Configuré un servidor SFTP seguro. Esto me permite subir y descargar archivos de forma remota y encriptada, funcionando como mi propia "nube" privada. Tuve que gestionar cuidadosamente los permisos de usuarios y grupos en Linux para asegurar que cada usuario solo pudiera acceder a su propia carpeta, protegiendo los archivos del sistema.

El reto más interesante fue la configuración de red para hacerlo accesible desde fuera de casa. Abrí los puertos necesarios en el router (Port Forwarding) y configuré un servicio de DNS Dinámico (No-IP). Gracias a esto, tengo un dominio fijo para conectarme a mi servidor desde cualquier lugar del mundo, aunque mi compañía de internet cambie mi dirección IP pública.

Tecnologías: Linux, SSH, SFTP, No-IP.
