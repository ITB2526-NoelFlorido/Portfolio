Detalles del proyecto:

Sobre el proyecto

Este proyecto consiste en la creación y administración de un curso completo en la plataforma Moodle. La temática elegida fue "Pokémon Competitivo", diseñado para enseñar estrategias avanzadas a jugadores novatos.


Tecnologías utilizadas

  • Moodle Cloud (SaaS)
  • Creación de usuarios y roles
  • Gestión de insignias y calificaciones
  • HTML/CSS básico para la personalización del tema

Resultados

Se logró implementar un sistema de aprendizaje funcional donde los alumnos podían registrarse, completar cuestionarios sobre IVs y EVs, y recibir retroalimentación automática.
